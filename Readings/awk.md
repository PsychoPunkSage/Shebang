# AWK Command

## Syntax

```bash
# 1. Way1
awk [options] 'pattern {action}' file_name

# 2. Way2
echo "Data" | awk [options] 'pattern {action}'
```

**`Options`**:<br>&nbsp;&nbsp;&nbsp;&nbsp;**`-F`** field seperator<br>&nbsp;&nbsp;&nbsp;&nbsp;**`-v`** var=value<br>&nbsp;&nbsp;&nbsp;&nbsp;**`-f`** file<br>
